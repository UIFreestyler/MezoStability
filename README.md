# MezoStability

## Welcome to the official mezo repository, here you can start by installing some basics to your desired folder:

- Sounddevice
- Numpy

### Via UV:
· uv pip install numpy sounddevice

### Via Pip:

· pip install numpy sounddevice

### Installation:

Git cloning is very easy, all you need is a shell and a brain.
Start by doing:

git clone https://github.com/UIFreestyler/MezoStability (you can do with depth=1 but doesnt matter)

### Change your directory to it:

cd MezoStability

### Now run the python file:

python3 mezo_audio.py && python3 mezo.py

### All the files are open to change! Some functions are still in development (Warning)
